Cíl webu

Docker je sada nástrojů, která nám umožňuje jasně definovat předpoklady pro správný běh a funkci jakékoliv aplikace. Předpoklady mohou být například: verze operačního systému, programovací jazyky, knihovny a závislosti. Na základě těchto závislostí nám Docker umožňuje vytvořit tzv. containery. Containery představují izolované prostředí pro běh aplikací. Využití Dockeru může výrazně zjednodušit vývoj, testování, běh a škálování aplikací.

Cílem by mělo být představit nástroj Docker a vytvořit řadu návodů a ukázek využití Dockeru při vývoji jakéhokoli softwaru.
